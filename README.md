# Exchange files between local and server for SAP

## Overview

This tool allows you to transfer files between the SAP application server and your local PC.

## Usage

Execute transaction code SE38 to register a new program with an arbitrary ID (e.g., Z_EXCHANGE_FILES).

You can run the program by copying and pasting the source code and activating it.

## Issues

Because there is hardcoded Japanese text, you will need to replace it with translated text when using it in an English environment.
